MySpace Clone

Piccolo social network ispirato a MySpace: registrazione, login e pagina profilo personalizzabile con bio e tema.

Cosa fa questo progetto

Questo repository contiene il codice di un semplice social network stile MySpace, sviluppato in PHP procedurale con database MySQL. È pensato come esercizio personale per:

  • gestire registrazione e autenticazione con password hashate
  • salvare i dati utente in un database MySQL
  • mostrare e modificare il profilo personale (bio, tema, ecc.)

Funzionalità principali

  • register.php - form di registrazione con controlli base
  • login.php - accesso con email e password
  • profile.php - pagina profilo dopo il login
  • logout.php - chiusura della sessione
  • delete_account.php - eliminazione del proprio account
  • config.php - connessione al database e gestione sessione
  • style.css - stile dark, ispirato ai social anni 2000

Stack & requisiti

Tecnologie usate
PHP 8.x (procedurale) MySQL / MariaDB HTML5 + CSS3 XAMPP / ambiente locale
Prerequisiti
  • Ambiente locale con Apache + PHP + MySQL (es. XAMPP)
  • Database MySQL chiamato myspace_clone
  • Tabella users creata con lo script SQL indicato nel repo

Come eseguirlo in locale

  1. Clona o scarica questo repository nella cartella del server locale (ad es. htdocs/myspace-clone in XAMPP).
  2. Crea un database MySQL chiamato myspace_clone.
  3. Esegui lo script SQL per creare la tabella users.
  4. Verifica le credenziali in config.php (host, user, password).
  5. Avvia Apache e MySQL, poi apri il browser su: http://localhost/myspace-clone/index.php
